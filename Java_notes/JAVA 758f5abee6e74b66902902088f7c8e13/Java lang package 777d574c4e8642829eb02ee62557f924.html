Object class
  • object class is the parent class for all the classes in java, in other words every class is directly or indirecty iherited from object class.
  • equals → will compare two objects reference and will return true if two references are holding the same object.
  • hashcode → every object created in java will have an unique code, compiler will assign the code, can be overrided as well
  • toString → can be overrided, if not overided it will print the packagename.classname@hashcode (example: roughuse.roughuse@6acbcfc0 )
  • notify(), wait() → cannot be overrided because it is declared with the final keyword.
Wrapper class
  • wrapper class is used to make the primitive data type ( Boolean, char, byte, int, short, long, float, and double) to class and we make object from the class.
  • wrapper classes are available for the following data type
    1. Character → separate child class inside object class.
    1. Boolean → separate child class inside object class.
    1. byte
    1. short
    1. integer
    1. long
    1. float
    1. double
    • object class is the main class and Number class is the child class of that and these(3-8) are child classes of Number class.
