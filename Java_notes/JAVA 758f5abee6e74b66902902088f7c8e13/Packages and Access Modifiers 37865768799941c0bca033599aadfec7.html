what is package?
  • a package is collection of classes or interfaces or other packages.
  • they are used for organizing java project.
  • basicallly if we want to organize our laptop documents we create a folder, tha’s is called package in java
  • organizing java files or class files in a single folder is called as package.
  • packages can be imported using import keyword

How to create packages?
  • create file and wite the codes.
  • using the following command in the first line of the program a package can be created
  • if I compile the program using normal compile method “javac Demo.java” in terminal, it will create the Demo.class in the same folder.
  • that’s not what i needed, Demo.class in a separate package/folder with the name mypack1, so the following command is used.
Syntax:
javac -d {path to save the package} classname.java

example:(saving the package in the same folder where the Demo.java file is.)
javac -d . Demo.java
  • where the dot(.) the path of the current folder.
  • any number of .class files can be stored in a package.
How to create subpackages?
  • after the main package name add a dot(.) and give the subpackage.
Syntax:
package mainPackageName.subPackageName;

example:
package mypack1.mypack2;
  • mypack2 subfolder will be created inside the mypack1 folder.
  • more number of subpackages can be added as well using this.

Access Modifiers
  • they are used for specifing the access levels or visiblity of members of a class.
  • there are 4 types of access modifiers
    1. default
    1. private
    1. protected
    1. public
  • a super/outer class can have members, method and inner class.
  • all these things are eligible to have anyone of the 4 access modifiers.
  • the super class can either have default or public and the super class can be used in two different ways.
  • 1st usage → used to create an object (term used: hasA).
  • 2nd usage → used to create a sub-class(term used: isA).
Overall view of accessiblity levels
  • default access modifiers are public but only inside a package, outside the package they are not accessible.
