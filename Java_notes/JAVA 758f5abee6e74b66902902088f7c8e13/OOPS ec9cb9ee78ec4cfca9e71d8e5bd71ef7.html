Class and Objects:
  • Anything in the world can be defined by their properties and behaviour

Example: for a TV the properties are channel, volume, colour, shape etc. the behaviour are change the channel, increasing or decreasing the volume etc.

  • Behaviour is written in the form of methods.
  • The radius and height are the properties of a cylinder.
  • The lidArea(),surfaceArea() and volume() are the methods(i.e.: Behaviour) of the cylinder.

Data hiding
  • Using access modifiers, the properties can be hidden
  • The following question will raise, the how do we initiate the values for the properties?
  • It is by using the get function and set function.

Example: for rectangle

  • The length can only be positive so if statement is used.
  • The same is used for breadth as well.
  • The set function is used to assign the actual values
  • The get function is used to print the value of the length when the method is called.
Types of properties
  • Three types:
    1. Read and writable property
    1. Read only
    1. Write only
  1. Read and write property
    • When both the set and get method are available then the property become read and writeable
    • Example: length of rectangle, mark of a student
  1. Read only property
    • Only get method will be there and set method is not allowed then that becomes read only property.
    • Example: student roll number(since there is no changing in the roll number of a student get method is enough), Bank account number as well.
  1. Write only property(rarely used)
    • Only set method will be used there and¬†method Is not allowed then that becomes write only property
    • Used between two programs
    • Example: one program will only read the data and another program will only write the data
    • Producer and consumer program
    • Producer ‚Üí set method is used in that program
    • Consumer ‚Üí get method is used in that program
Constructor
  • Values are preset when creating an object(construction of an object) of a class is called as constructor.
  • Definition: Constructor is a method of a class which is automatically called whenever an object is created.
  • Every class in java has a default constructor

How to write a constructor?

  • Constructor will have same name as class name
  • Constructor will not have any return type, not even void will be there.
  • Constructor is usually public
  • if constructor is private then objects cannot be created.
