Inner Class/Nested class
  • Used to reduce the complexity of a class
  • If a class is so big, then an inner class can be defined to make it small.
  • Interface can also be defined.
  • There are 4 types of inner class:
    1. Nested inner class
    1. Local inner class
    1. Anonymous inner class
    1. Static inner class
Nested inner class:
  • Inner class can access the member of outer class.
  • Inside the outer class object of inner class can be created.
  • Reference variable is needed to access the identifiers in the inner class.
  • Outer class can access the members/ identifiers of inner class directly, but inner class can be accessed by creating an object.
  • Every class is created in separate file in java.
  • Outer class → class, inner class → outer$inner.class
    • Outer.Inner i = new Outer().new Inner(); → used to access the inner class variable in the main function

  • classes inside the method which are local to the outer class.

Anonymous Inner class
  • When an abstract class is created, it has to override the methods in that class to make the new class concrete or make the new class abstract to run the program without error.

The method inside the abstract class can be overridden without creating a new class.

  • after creating the object, the abstract class is overridden with in { } → this will create an anonymous class during compilation.
  • Interfaces can also be used in the same manner.
  • This is used when the method has little usage, there is no need to create separate class and method.
Static Inner class
  • static inner classes are the static members of the outer class.
  • objects of static inner class can be created outside the outer class, so creating outer class object is not necessary.
  • unlike nested inner class, the object of static inner class can be directly declared outside the outer class, without creating the object of outer class.
  • static inner class members can only access the members which are static in the outer class, it cannot access non static member.
