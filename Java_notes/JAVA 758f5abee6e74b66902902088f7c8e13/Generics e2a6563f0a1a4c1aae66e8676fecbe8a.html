Example of Generics in a single object and Array of object

code and ouput are given down below:

package generics;
class Data<T>
{
        private T obj;
        public void set(T v) {obj = v;}

        public T get(){return obj;}
}
class MyArray<T>
{
    private T data[] = (T[])new Object[10];
    private int length=0;
    void append(T value){data[length++] = value;}

    void display(){for(int i = 0; i<length; i++) System.out.println(data[i]);}
}
public class Generics {
    public static void main(String[] args)
    {
        Data<Integer> d= new Data<>();
        d.set(10);
        System.out.println("Data Class: ");
        System.out.println(d.get());

        System.out.println("\nMy array Class: ");
        MyArray<Integer> my = new MyArray<>();
        my.append(10);
        my.append(20);
        my.append(30);
        my.display();
    }
}

what wil l happen when No generics parameter are given

  • if no generics parameters(MyArray my = new MyArray<>();) are given then the my becomes a simple object reference and any input can be given while calling append method.
  • warning will be produced but error will not be there.
what will happen if more number of parameter is passed in the generic
  • In java more number of generic can be passed.
  • when creating an generic object parameter it should be equal to the generic class parameter .
  • Usually this is done in case of Key-value pair i.e.: <K,V> (Another Topic).
what about subclass(subtypes)
  • If one class is generic, then its child class is also generic class and can allow only the datatype of the generic parent class.
  • example: consider Myarray is parent array and Myarray2 is the subclass of parent array.
  • if (Myarray<String>) has type String then Myarray2 can allow only strings.
  • instead of that Myarray<String> if Myarray<T> is placed then Myarray2 subclass has to be made generic as well by doing this Myarray2<T> .
